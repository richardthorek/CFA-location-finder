Application Overview

The CFA Location Finder is a web application that displays real-time fire alerts from the Country Fire Authority (CFA) on an interactive map. It automatically parses alert messages to extract locations in Victoria, Australia, and displays them with markers on a MapBox map.

Quick Setup

  1. Get a free MapBox API token from account.mapbox.com
  2. Update the token in app.js
  3. Deploy to Azure Static Web Apps (connects via GitHub)
  4. The Azure Function automatically handles CFA feed fetching

Technology Stack

  • Frontend: Pure HTML, CSS, JavaScript (no framework dependencies)
  • Maps: MapBox GL JS v2.15.0
  • Backend: Azure Functions (Node.js)
  • Hosting: Azure Static Web Apps
  • Deployment: GitHub Actions (automated)

Next Steps

To deploy this application:

  1. Fork or clone the repository
  2. Set up a MapBox account and get your API token
  3. Create an Azure Static Web App and connect to GitHub
  4. Configure the deployment token as a GitHub secret
  5. Push to main branch - GitHub Actions will handle deployment
